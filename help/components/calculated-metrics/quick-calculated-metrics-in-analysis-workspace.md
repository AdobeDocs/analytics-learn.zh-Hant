# 分析工 [!UICONTROL 作區中的] 「快速計算量度」 {#quick-calculated-metrics-in-analysis-workspace}

現在，您可以在 [!UICONTROL 分析工作區中] ，即時建立計算量度。  如果您想要在自由表格中快速變更百分比或劃分 [!UICONTROL 度量] ，只需從標題中選取度量  ，然後按一下滑鼠右鍵並選取您想要的運算子。  新的 [!UICONTROL 計算量度] 將出現在右側。
