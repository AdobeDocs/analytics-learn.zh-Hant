# 在分段中使用「等於任一」運算子

區段產生器中已新增「等於任一」和「不等於任一」區段運算子。 使用這些運算子來篩選特定項目的清單，而不是使用多個OR陳述式或「包含任何項目」。 使用這些運算子而非&#39;contains any of&#39;也有助於改善查詢速度。
