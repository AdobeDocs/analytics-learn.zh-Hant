# [!UICONTROL 分段中的維度模型]

[!UICONTROL 維度]模型可透過分段中任何[!UICONTROL 維度]的齒輪提供。這些模型主要有助於循序分段，並可讓您對[!UICONTROL 維度] (預設) 的持續值、執行個體或[!UICONTROL 維度]的不重複執行個體進行分段。
