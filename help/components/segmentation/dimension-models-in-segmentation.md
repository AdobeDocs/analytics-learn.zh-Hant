# [!UICONTROL 分段中的維度模型]

維度模型可透過分段中任何維度的齒輪提供。這些模型主要有助於循序分段，並可讓您對維度 (預設) 的持續值、執行個體或維度的不重複執行個體進行分段。如需詳細檔案，請造訪 [區段產生器](https://experienceleague.adobe.com/docs/analytics/components/segmentation/segmentation-workflow/seg-build.html).
