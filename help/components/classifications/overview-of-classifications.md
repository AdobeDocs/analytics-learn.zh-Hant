# Adobe Analytics中的[!UICONTROL 分類]概觀 {#overview-of-classifications-in-adobe-analytics}

 分類是將中繼資料（描述性屬性）新增至產品、行銷活動、頁面、客戶等的強大方式。這些屬性可讓您將類似的元素分組並劃分資料，深入探討哪些元素對您有用，哪些不適用，借此協助您更清楚了解行銷和體驗工作。
