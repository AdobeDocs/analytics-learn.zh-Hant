# 透過 SFTP 傳送 Data Warehouse 請求

Data Warehouse 指的是儲存和自訂報表所使用的 Analytics 資料複本，您可以透過篩選資料來執行這些資料。 您可以要求報表依據您的特定問題，從原始資料顯示進階資料關係。Data Warehouse 報表會以電子郵件方式或透過 FTP 傳送，處理時間最長需要 72 小時。處理時間依查詢的複雜度以及請求的資料量而定。
