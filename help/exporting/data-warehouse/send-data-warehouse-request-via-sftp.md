# 透過 SFTP 傳送 Data Warehouse 請求

資料倉庫是指儲存和自定義報告的分析資料副本，您可以通過篩選資料來運行這些資料。 您可以要求報表依據您的特定問題，從原始資料顯示進階資料關係。Data Warehouse 報表會以電子郵件方式或透過 FTP 傳送，處理時間最長需要 72 小時。處理時間依查詢的複雜度以及請求的資料量而定。
