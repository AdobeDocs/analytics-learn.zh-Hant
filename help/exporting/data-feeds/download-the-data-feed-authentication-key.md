# 下載資料摘要驗證金鑰

建立資料摘要，Adobe 就能知道要將原始資料檔案傳送至何處，以及您想在每個檔案中納入哪些資料。 Analytics 中有提供 SFTP 對資料摘要的支援。 必須填入 SFTP 主機、用戶名稱和目的地站點，才能包含有效的 RSA 或 DSA 公開金鑰。 建立摘要時，您可以下載相關的公開金鑰。
