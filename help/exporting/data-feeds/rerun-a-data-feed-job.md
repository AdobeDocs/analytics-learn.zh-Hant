# 重新執行資料摘要工作

Adobe Analytics中的資料摘要UI可大幅簡化資料摘要管理工作。 如果您具有管理資料摘要的許可權，可以自行在資料摘要UI中重新執行資料摘要工作，而不需要客戶支援團隊重新傳送資料摘要檔案。
