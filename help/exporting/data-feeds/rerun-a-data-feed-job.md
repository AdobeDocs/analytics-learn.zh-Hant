# 重新執行資料摘要工作

Adobe Analytics中的資料摘要UI讓資料摘要管理更輕鬆。 如果您有管理資料摘要的存取權，可以使用資料摘要UI自行重新執行資料摘要工作，而不需要客戶支援團隊重新傳送最近的資料摘要檔案。
