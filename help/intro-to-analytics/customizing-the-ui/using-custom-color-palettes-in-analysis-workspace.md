# 在分析工 [!UICONTROL 作區中使用] 「自訂色盤」 {#using-custom-color-palettes-in-analysis-workspace}

您可以變更 [!UICONTROL Analysis Workspace中使] 用的色盤  ，以更符合您的品牌或客戶的品牌。

>[!VIDEO](https://video.tv.adobe.com/v/23876/?quality=12)
