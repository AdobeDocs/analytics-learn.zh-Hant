# 使用Experience Cloud除錯程式檢視後處理的點擊

Experience Cloud除錯程式現在提供後處理資料。 透過Experience Cloud登入並開啟「擷取後處理的點擊」，以檢視已套用VISTA和處理規則後的點擊。 例如，如果您的組織嚴重依賴處理規則中的上下文資料對應，您現在可以看到每個上下文資料變數直接儲存在除錯程式中的位置。
