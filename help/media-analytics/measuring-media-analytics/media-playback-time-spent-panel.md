# 媒體播放時間面板

媒體播放時間花費面板透過一天中所選粒度期間的觀看時間，來讓媒體用戶了解他們的觀看量。您也可以「放大」一天中的一段期間，或建立循序時間區段，以清楚顯示一天中各部分進出的潛在客戶。
