# 將使用者資產轉移給不同的使用者

「使用者管理」頁面上的「管理報表使用者和群組」可讓您管理使用者和群組，以及控制對報表、工具和報表套裝的存取。您可以透過Adobe Analytics中的使用者管理，將控制面板、日曆事件、書籤等使用者帳戶項目，從一個使用者帳戶指派給另一個使用者的帳戶。
