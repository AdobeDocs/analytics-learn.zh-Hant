# 將使用者資產轉移給其他使用者

「使用者管理」頁面上的「管理報表使用者和群組」可讓您管理使用者和群組，以及控制對報表、工具和報表套裝的存取。 您可以透過 Adobe Analytics 中的使用者管理將某個使用者帳戶中使用者帳戶項目 (例如儀表板、行事曆活動、書籤等) 指派給另一個使用者的帳戶。
