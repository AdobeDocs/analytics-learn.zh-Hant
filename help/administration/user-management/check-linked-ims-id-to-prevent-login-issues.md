# 檢查連結的IMS ID以防止登入問題

Adobe Analytics中的連結IMS ID不正確或遺失，可能會導致使用者的登入問題。 使用者可自行檢查連結的IMS ID，並聯絡客戶支援以了解任何問題，免除執行常見疑難排解步驟的麻煩。
