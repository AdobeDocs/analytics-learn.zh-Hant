# 檢查綁定的 IMS ID 以防止登入問題

Adobe Analytics 中綁定的 IMS ID 不正確或遺失可能導致使用者發生登入問題。 使用者可以自行檢查綁定的 IMS ID，並在發生任何問題時聯絡客戶支援，以省去執行常見疑難排解步驟的麻煩。
