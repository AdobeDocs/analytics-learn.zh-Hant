# 檢查綁定的 IMS ID 以防止登入問題

Adobe Analytics中連結的IMS ID不正確或缺失可能會導致用戶登錄問題。 用戶可以自己檢查連結的IMS ID並聯繫客戶支援瞭解任何問題，從而省去執行常見故障排除步驟的麻煩。
