# 啟用可選時間戳記設定

「可選時間戳記」可讓我們在相同的全域報表套裝中混合時間戳記與非時間戳記資料、從行動應用程式傳送時間戳記資料至全域報表套裝，以及將應用程式升級，以便使用離線追蹤而不需建立新的報表套裝。
