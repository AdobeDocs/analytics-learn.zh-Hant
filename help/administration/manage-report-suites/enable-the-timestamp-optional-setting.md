# 啟用時間戳記選用設定

可選時間戳記可讓我們在相同的全域報告套裝中混用有時間戳記和沒有時間戳記的資料、從行動應用程式傳送有時間戳記的資料給全域報告套裝，並升級應用程式以使用離線追蹤，而不必建立新的報告套裝。
