# Analytics 管理員記錄概觀

記錄檔可協助您查看使用者何時登入、其使用情形、存取權、報表套裝以及管理員變更。 管理記錄檔會報告管理員在管理工具中所做的一切變更。 此記錄提供從三個記錄中的任何一個存取用戶定義的報表的管道。
