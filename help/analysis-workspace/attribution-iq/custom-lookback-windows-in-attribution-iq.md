# Attribution IQ 中的自訂回顧期間

自訂回顧期間可讓您將歸因期間擴展至報表範圍以外 (最多 90 天)，並套用至報表範圍內的每個轉換。此選項一般會說明在前幾個月發生的互動，提高在報告期間初期所發生事件的歸因準確性。
