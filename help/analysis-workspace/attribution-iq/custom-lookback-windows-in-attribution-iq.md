# Attribution IQ 中的自訂回顧期間

自訂回顧期間可讓您將歸因期間擴展至報表範圍以外 (最多 90 天)，並套用至報表範圍內的每個轉換。此選項通常會計算前一個月的互動，以提高報表期間前期所發生之事件的歸因準確度。
