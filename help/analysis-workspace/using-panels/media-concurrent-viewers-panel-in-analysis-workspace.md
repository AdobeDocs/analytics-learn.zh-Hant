# Analysis Workspace 中的「媒體同時檢閱者」面板

如果是 Media Analytics 客戶，「媒體同時檢閱者」面板可讓您了解發生高峰期同時觀看人數或客戶流失的時間，以針對內容品質和檢視者參與度提供重要解析，並有助於疑難排解或完成數量/規模的相關規劃。
