# 使用同類群組表進行客戶流失分析 {#churn-analysis-with-cohort-tables}

[!UICONTROL 同類] 群組表格提供兩種分析類型：  保留 [!UICONTROL 和流失率]。雖然[!UICONTROL 保留]顯示一段時間內保留了多少使用者，但[!UICONTROL 流失率]選項將描繪遺失多少使用者（1 — 保留）。
