# Analysis Workspace 中的組合圖表

了解 Analysis Workspace 中的組合圖表視覺效果。組合圖表視覺效果可讓您快速建立比較視覺效果，而不用先建置表格。您可以在折線/條形組合中輕鬆地檢視資料中的趨勢。
