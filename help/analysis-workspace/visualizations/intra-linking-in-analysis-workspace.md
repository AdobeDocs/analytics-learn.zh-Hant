# 分析工作區中的內部連結 {#intra-linking-in-analysis-workspace}

內部連結可讓分析人員在分析工作區中建立專案，將專案的讀者導向專案內的精確面 [!UICONTROL 板] 和視覺化。

>[!VIDEO](https://video.tv.adobe.com/v/23724/?quality=12)

內部連結也可用來建立目錄，並整體呈現更有效的資料故事。 由於新的超連結功能，現在也可以進行跨專案連結。
