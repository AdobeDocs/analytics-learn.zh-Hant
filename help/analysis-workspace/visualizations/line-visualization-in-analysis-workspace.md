# Analysis Workspace 中的線條視覺效果

「線條」視覺效果可描繪某個趨勢時段內的量度。視覺效果設定可讓您自訂每個線條的視覺外觀，並加入「顯示 X/Y 軸」、「顯示最小/最大標籤」和「顯示趨勢線」的設定。
