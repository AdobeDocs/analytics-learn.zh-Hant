# 分析工作區中的線條視覺化

「行」視覺化會描述趨勢時段內的量度。 視覺化設定可讓您自訂每行視覺化的外觀，並包含「顯示X/Y軸」、「顯示最小／最大標籤」和「顯示趨勢線」的設定。
