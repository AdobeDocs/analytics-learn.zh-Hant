# 使用 Workspace 中的元件下拉選單

有了元件下拉選單，您就能從相容元件的清單中挑選項目，以建置您的分析。 如果您找不到自己所需的項目，您也可以直接從下拉式選單建立新的元件。 如果您有更進階的使用案例，這些空間也會繼續做為下拉區域作用。
