# 在Analysis Workspace中使用[!DNL Freeform Table Builder]

[!DNL Freeform Table Builder] 可讓您拖放許多維度、劃分、量度和區段，以建立可回答更複雜業務問題的表格。資料不會即時更新；而是會在您按一下「建置」後更新，因為表格只更新一次，所以可為您節省時間。
