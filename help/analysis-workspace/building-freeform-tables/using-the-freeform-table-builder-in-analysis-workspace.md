# 在 Analysis Workspace 中使用自由格式表格產生器

[!DNL Freeform table builder]可讓您拖放使用許多維度、劃分、量度和區段，方便建置可回答更複雜商業問題的表格。資料將不會立即更新；由於表格僅會更新一次，因此在您按一下「建置」後資料將會更新，為您節省時間。
