# 在分析工 [!DNL Freeform Table Builder] 作區中使用

[!DNL Freeform Table Builder] 可讓您拖放許多維度、劃分、量度和區段，以建立可回答更複雜商業問題的表格。 資料不會即時更新；而是在您按一下「建置」後，它會更新，因為表格只更新一次，所以節省您的時間。
