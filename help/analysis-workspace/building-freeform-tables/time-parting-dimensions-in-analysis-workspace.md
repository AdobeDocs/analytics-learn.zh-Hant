# [!DNL Time-parting] Analysis Workspace 中的維度 {#time-parting-dimensions-in-analysis-workspace}

[!DNL Time-parting]功能會擷取所收集的點擊的時間戳記，並採用更有意義的維度加以分段，例如「小時」或「星期」。這些維度可在 Analysis Workspace 中使用。
