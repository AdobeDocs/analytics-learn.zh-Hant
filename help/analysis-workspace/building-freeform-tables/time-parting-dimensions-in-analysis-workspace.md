# [!DNL Time-parting] Analysis Workspace 中的維度 {#time-parting-dimensions-in-analysis-workspace}

[!DNL Time-parting] 取用所收集點擊的時間戳記，並將其劃分為更有意義的維度，例如「小時」或「星期」。這些維度現在已可在Analysis Workspace中立即使用。
