# 在 Analysis Workspace 中重新排序靜態資料列

您可以在自由格式表格產生器中手動重新排序靜態資料列。這可讓您依照您所需的確切順序排列靜態資料列 - 例如，造訪次數、然後是訂單，再來是轉換率。
