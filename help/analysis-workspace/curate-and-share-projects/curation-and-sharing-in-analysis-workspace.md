# [!UICONTROL Analysis Workspace 中的「組織」與「共用」] {#curation-and-sharing-in-analysis-workspace}

在Analysis Workspace中建置專案後，您就可以[!UICONTROL 組織]（簡化）專案，並與組織中的其他團隊和個人共用專案，讓他們能夠探索專案，並自行提供資料。 此影片會示範如何使用[!UICONTROL 組織]和共用。
