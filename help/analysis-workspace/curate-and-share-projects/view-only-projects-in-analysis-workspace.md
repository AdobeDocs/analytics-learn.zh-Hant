# Analysis Workspace 中僅限檢視的專案

Workspace 專案能僅以「可供檢視」狀態與使用者共用。當「檢視」收件者開啟共用專案時，獲得的專案體驗限制較嚴格，除了沒有左邊欄，互動也會受限。
