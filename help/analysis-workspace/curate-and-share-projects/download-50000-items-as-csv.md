# 將 50,000 個項目下載為 CSV 檔

「以CSV格式下載項目」選項可讓您以自由格式表格下載一個維度的最多50,000個項目，並套用區段和篩選器。 雖然 UI 在分頁前會持續顯示最多 400 個項目，但此選項可讓您在 Analysis Workspace 以外存取更多資料列。
