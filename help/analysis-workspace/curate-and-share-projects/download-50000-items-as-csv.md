# 將 50,000 個項目下載為 CSV 檔

「將項目下載為 CSV 檔」選項可讓您現在可以在自由格式表格中最多下載一個維度的 50,000 個項目，並套用區段和篩選條件。 雖然 UI 在分頁前會持續顯示最多 400 個項目，但此選項可讓您在 Analysis Workspace 以外存取更多資料列。
