# 在左側欄中使用篩選器（標籤）

篩選（或標籤）是快速且簡單的方式，可在分析工作區中篩選左側邊欄，以尋找所需 [!UICONTROL 的元] 件。 運用Adobe提供的現成可用標籤，或新增您自己的自訂標籤以組織相關 [!UICONTROL 元件]。
