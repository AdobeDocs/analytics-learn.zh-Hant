# 使用多選下拉式篩選器

瞭解Analysis Workspace中在下拉式篩選器中多選欄位的選項，該選項可讓使用者顯示任何選定欄位的資料。 如需詳細檔案，請造訪 [面板概觀](https://experienceleague.adobe.com/en/docs/analytics/analyze/analysis-workspace/panels/panels#static-drop-down-segments).
