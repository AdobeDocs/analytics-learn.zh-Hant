# Analysis Workspace 中的註解

Workspace 中的註解讓您能夠有效地將內容相關的資料細微差別和深入解析傳達給整個組織內的用戶。您可以用已知的資料問題、國定假日、行銷活動推出等來標註日期或日期範圍，以便更有效告訴用戶他們所見線性圖、表格等的原因。
