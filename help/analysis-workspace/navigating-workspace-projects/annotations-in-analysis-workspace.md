# Analysis Workspace注釋

Workspace中的注釋使您能夠有效地將上下文資料細微差別和洞察力傳達給整個組織的用戶。 您可以用資料問題、國定假日、行銷活動推出等來標註日期或日期範圍。以便更好地告知用戶，他們為什麼在折線圖、表格等中看到了什麼。
