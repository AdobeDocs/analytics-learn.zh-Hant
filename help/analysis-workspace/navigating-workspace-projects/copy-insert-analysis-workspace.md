# Analysis Workspace 中的複製並插入 {#copy-and-insert-in-analysis-workspace}

複製 + 插入功能可讓您將一個面板或專案的視覺效果複製到另一個，或是橫跨專案複製完整面板。

>[!VIDEO](https://video.tv.adobe.com/v/23230/?quality=12)

使用此功能將您的分析組織成邏輯資料故事和專案，或從範本複製以取得起步優勢。
